1. Using System.Diagnostics.Process to Call yt-dlp (Recommended)

What it does: This method involves executing yt-dlp as an external process using System.Diagnostics.Process. This is the most common approach for invoking command-line tools from a WPF app.

How it works:

    You run yt-dlp from your WPF application by creating a new process and passing the necessary command-line arguments.
    You can capture the output and errors in your WPF UI to provide real-time feedback to users (e.g., download progress).

Example Code:

using System;
using System.Diagnostics;
using System.Windows;

public partial class MainWindow : Window
{
    public MainWindow()
    {
        InitializeComponent();
    }

    private void DownloadButton_Click(object sender, RoutedEventArgs e)
    {
        // Set the video URL and yt-dlp executable path
        string videoUrl = "https://www.youtube.com/watch?v=dQw4w9WgXcQ";
        string ytDlpPath = @"C:\path\to\yt-dlp.exe"; // Update with correct path

        // Create process to run yt-dlp
        Process process = new Process();
        process.StartInfo.FileName = ytDlpPath;
        process.StartInfo.Arguments = $"-f best -o \"{System.IO.Directory.GetCurrentDirectory()}\\%(title)s.%(ext)s\" {videoUrl}";
        process.StartInfo.RedirectStandardOutput = true;
        process.StartInfo.RedirectStandardError = true;
        process.StartInfo.UseShellExecute = false;
        process.StartInfo.CreateNoWindow = true;

        // Handle output and error
        process.OutputDataReceived += (sender, args) =>
        {
            if (args.Data != null)
            {
                Console.WriteLine(args.Data); // You can update the UI with this
            }
        };
        process.ErrorDataReceived += (sender, args) =>
        {
            if (args.Data != null)
            {
                Console.Error.WriteLine(args.Data); // Error output from yt-dlp
            }
        };

        // Start the process and capture output
        try
        {
            process.Start();
            process.BeginOutputReadLine();
            process.BeginErrorReadLine();
            process.WaitForExit(); // Wait for the process to finish
        }
        catch (Exception ex)
        {
            MessageBox.Show($"Error: {ex.Message}");
        }
    }
}

Why this works: Process.Start runs the yt-dlp command in a new process and allows you to pass arguments such as the URL of the video to download. You can also capture real-time output to provide progress feedback.
